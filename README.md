# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md) uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

## Technologies Used

This project is built with a modern tech stack that includes the following frameworks, libraries, and tools:

### Frameworks

- **React:** A JavaScript library for building user interfaces.

### Libraries

- **Axios:** A promise-based HTTP client for the browser and Node.js.
- **Framer Motion:** A production-ready motion library for React.
- **Leaflet:** An open-source JavaScript library for mobile-friendly interactive maps.
- **React Beautiful DnD:** A library for creating beautiful and accessible drag-and-drop lists.
- **React Hot Toast:** A lightweight and customizable toast notification library.
- **React Icons:** A collection of popular icon libraries for React.
- **React Intersection Observer:** A React component for observing changes in the intersection of a target element with an ancestor element or with a top-level document's viewport.
- **React Leaflet:** React components for Leaflet maps.
- **React Query:** A library for fetching, caching, and updating data in React applications.
- **React Router DOM:** A library for routing in React applications.
- **React Slick:** A carousel component built with React.
- **React Toastify:** A notification library for React applications.
- **Slick Carousel:** A responsive carousel jQuery plugin.
- **Tailwind CSS:** A utility-first CSS framework for rapidly building custom designs.

### Build Tools

- **Vite:** A fast and opinionated build tool for modern web development.

### Linting

- **ESLint:** A pluggable and configurable linter tool for identifying and reporting on patterns in JavaScript.
